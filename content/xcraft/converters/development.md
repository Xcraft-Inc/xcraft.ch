To run test:

- `npm test xcraft-core-converters`

To debug test:

- Run [Visual Studio Code][1]
- Open the file `xcraft-core-converters/lib/\*.js` or
  `xcraft-core-converters/test/\*.js` to debug
- Set a breakpoint
- Run "Test current file (mocha)"

[1]: https://code.visualstudio.com/
